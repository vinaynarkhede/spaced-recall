DECK: Networking::linux_namespaces
QUESTION: What is the complete hierarchy of a network call from browser to destination?
ANSWER: 1) Application (browser). 2) Socket layer (file descriptor). 3) Transport layer (adds source/dest ports). 4) Network layer (adds source/dest IPs). 5) Routing decision (check routing table). 6) Interface selection. 7) Data link layer (adds MAC addresses). 8) Physical layer (transmits).
